# 11ty

## Running the application locally

You need to have node, minimum version 18:
``` shell
node --version
```

#### Install dependencies

```shell
$ npm install
```

#### Run application

```shell
$ npm start
```

The website runs and can be accesed here [localhost:8080](http://localhost:8080)
